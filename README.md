##node 博客系统

####开始
>初始化
  
`npm install;//安装依赖`
>运行

`npm start;//运行`

####目录结构

* db  ---- 数据库存储目录
* models  ---- 数据库模型文件目录
* node_modules ---- node模块目录
* public ---- html,css,image公共文件目录
* routers ---- 路由文件目录
* schemas ---- 数据库结构文件（schema）目录
* views ---- 模板视图文件
* app.js ---- 引用启动文件
* package.json ---- 项目配置信息文件
